Detailed Description

Namespace for Newtonian Interaction related functions.

In the special applications of N-body simulations with Newtonian gravity, the Newtonian acceleration functions should be defined. The corresponding timescale to determing the next integrating step should also be provided. This is the namespace where all these functions are provided.

Newtonian acceleration and \(\partial W_{ij}/\partial \mathbf{x}_i\) from particle j to particle i (function type of ARC::pair_AW)

Parameters
[out]AijNewtonian acceleration vector for i particle from particle j. \(Aij[1:3] = m_i m_j xij[1:3] / |xij|^3 \).
[out]PijNewtonian potential of i from j. \( Pij = -m_i m_j /|xij| \)
[out]pWijTTL time transformation function partial derivates (component from j to i) \(\partial W_{ij}/\partial \mathbf{x}_i\) (used for TTL method). \(pWij[1:3] = mm_{ij} xij[1:3] /|xij|^3 \). (Total value is \(\frac{\partial W}{\partial \mathbf{x}_i} = \sum_{j} mm_{ij} \mathbf{x}_{ij}/|\mathbf{x}_{ij}|^3\))
[out]WijTTL time transformation function component with i,j (used for TTL method) \(Wij = mm_{ij} /|xij|^3\) total value is \( W = \sum_{i<j} mm_{ij} /|xij| \)
[in]xijrelative position vector [1:3] \( \mathbf{x_j} - \mathbf{x_i} \)
[in]miparticle i mass.
[in]mjparticle j mass.
[in]parsNewtionian_pars type data with members:
  • mm2: smooth mass coefficient \( \sum_{i<j} m_i m_j /(N(N-1)/2) \) (can be calculated by calc_mm2());
  • epi: adiustable parameter. 1) If epi>0: if \(m_i m_j < epi mm2\): \( mm_{ij} = mm2\) else: \( mm_ij = 0\) 2) If epi<0: \(mm_{ij} = m_i m_j\).
Returns
status: 0 for normal cases; 1 for the case when two particles have same positions
