Detailed Description

template<class int_par>
class ARC::chainpars< int_par >

The chain parameter controller class.

This class control the acceleration function and time transformation function (ARC::chainpars.pair_AW, ARC::chainpars.pair_Ap), integration methods and error parameters for chain, and timescale calculator ARC::chainpars.pair_T. The template depended type int_par is user-defined class for two-body interations used in ARC::chainpars.pair_AW, ARC::chainpars.pair_Ap and ARC::chainpars.pair_T

§ chainpars() [1/2]

template<class int_par>
ARC::chainpars< int_par >::chainpars ( )
inline

constructor with defaulted parameters

  • Acceleration function use ARC::Newtonian_AW() and ARC::Newtonian_Ap().
  • ARC method use logarithmic Hamiltonian (logH) (#alpha = 1.0, #beta = 0.0 #gamma = 0.0).
  • Phase/energy error limit exp_error = 1e-10.
  • Minimum physical time step dtmin = 5.4e-20.
  • Time synchronization error limit dterr = 1e-10.
  • Maximum extrapolation sequence index (accuracy order/iteration times) #exp_itermax = 20
  • The maximum sequence index (iteration times) is adjusted by error criterion
  • Bulirsch & Stoer sequence {h, h/2, h/3, h/4, h/6...} is used
  • No auto-step

[in]ext_sequence1: Romberg sequence {h, h/2, h/4, h/8 ...}; 2: Bulirsch & Stoer (BS) sequence {h, h/2, h/3, h/4, h/6, h/8 ...}; 3: 4k sequence {h, h/2, h/6, h/10, h/14 ...}; others: Harmonic sequence {h, h/2, h/3, h/4 ...} (defaulted 2. BS sequence)

Get auto-step parameters.

Parameters
[in]optionauto-step method
  • 0. no auto-step
  • 1. use extrapolation error to estimate next step
  • 2. use min( \(X/(gV)\), \(V/(gA)\)) to estimate next step
  • 3: use maximum sequence index to control next step. If index >iter_max, modification factor = factor_min; if index < iter_min, modification factor = factor_max;
  • 4: use mimimum user-defined two-body timescale of every two neigbor members to estimate next step
[in]factor_minif option = 1->3, it is the minimum step reduction factor (defaulted: 0.7)
[in]factor_maxif option = 1->3, it is the maximum step reduction factor (defaulted: 1.3)
[in]epscoefficient. If option = 2 or 4, coefficient is the multiplying factor for the estimated next step (defaulted: 0.125)
[in]iter_minif option = 3, it is the minimum sequence index (iteration times) criterion during extrapolation intergration (defaulted: 5)
[in]iter_maxif option = 3, it is the maximum sequence index (iteration times) criterion during extrapolation intergration (defaulted: 17)
